# Настройка Telegram Бота для Заказов (Express Pizza)

Данная инструкция поможет вам настроить получение заказов с сайта прямо в ваш Telegram.

## Шаг 1: Создание Бота
1. Откройте приложение Telegram и найдите пользователя **@BotFather**.
2. Отправьте ему команду `/newbot`.
3. Придумайте имя для бота (например, "Заказы Express Pizza").
4. Придумайте username для бота (должно заканчиваться на `bot`, например, `express_pizza_minsk_bot`).
5. @BotFather пришлет вам сообщение с вашим **токеном (Token)**. Он выглядит примерно так: `123456789:ABCDefghIJKLmnopQRSTuvwxYZ`.
   *Скопируйте этот токен - он понадобится на Шаге 3.*

## Шаг 2: Узнать свой ID (Chat ID)
1. Найдите в Telegram бота **@userinfobot** или **@GetIDs Bot**.
2. Отправьте ему любое сообщение или нажмите "Start".
3. Бот ответит сообщением, где будет ваш **ID (Id)**. Он состоит только из цифр, например `987654321`.
   *Скопируйте этот ID.*
   *Примечание: Если вы хотите, чтобы заказы приходили в группу, добавьте бота в эту группу и узнайте её ID.*

## Шаг 3: Добавление данных на сервер
1. Откройте файл `server/.env`.
2. Добавьте переменные:
   ```dotenv
   TELEGRAM_BOT_TOKEN=YOUR_BOT_TOKEN_HERE
   TELEGRAM_CHAT_ID=YOUR_CHAT_ID_HERE
   ```
3. Замените `YOUR_BOT_TOKEN_HERE` на **токен** из Шага 1.
4. Замените `YOUR_CHAT_ID_HERE` на **ID** из Шага 2.
5. Перезапустите backend (`npm run dev` в папке `server`).

Теперь уведомления отправляются сервером через `server/src/services/telegramService.js`, а клиентский stub удалён как устаревший.
